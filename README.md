# MoreMultilib
Simple multi-language library in C, C++, Rust and Python basic and advanced


## Supported Languages

- **C**
- **C++**
- **Python**
- **Rust**


## Project Structure

The project is organized into the following directories:

```
project-root/
│
├── c/
│ └── src/
│   ├── basic/
│   ├── advanced/
│   └── dp/
│
├── cpp/
│ └── src/
│   ├── basic/
│   ├── advanced/
│   └── dp/
│
├── python/
│ └── src/
│   ├── basic/
│   ├── advanced/
│   └── dp/
│
└── rust/
└── src/
  ├── basic/
  ├── advanced/
  └── dp/

```

--- 

Note: Python has more features
